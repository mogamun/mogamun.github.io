# 메타프롬프트: Vibe to Spec 앱 개발 기획 변환기

## 개요
비개발자의 추상적 아이디어("Vibe")를 개발자 친화적 앱 기획서로 변환하는 메타프롬프트 방법론. tilnote.io에 공유된 프롬프트 템플릿으로, 5가지 AI 페르소나와 6단계 워크플로우 정의.

## 핵심 내용
- **5가지 AI 페르소나**: 공감적 경청자, 명확한 질문자, 전문 번역가, 시스템 설계자, 선제적 제안자
- **6단계 워크플로우**: 아이디어 파악 → 기능 구체화 → 데이터/프로세스 탐색 → UX/UI 컨셉 도출 → 기술 고려사항 확인 → 반복/검증
- **출력물 구조**: 앱 개요, 타겟 사용자, 기능 명세(사용자 스토리), 아키텍처, UI/UX, 기술 스택, 비기능적 요구사항
- **실제 적용 사례**: 크로스워드 퍼즐 PDF 생성 웹앱 기획서 포함
- **고급 프롬프팅 기법**: CoT, 역할 프롬프팅, Few-Shot, 자기 비판/수정, 제로샷 유도 금지

## 원문 인용
> "사용자의 'Vibe'를 다음을 포함하는 개발자 친화적인 앱 기획 문서로 구체화하고 체계화한다."
